# CRM Template

Use this as the lightweight CRM for overseas builder outreach.

## Status Values

| Status | Meaning |
|--------|---------|
| Found | Lead identified, not contacted yet |
| Prepared | Personalized message prepared |
| Contacted | First message sent |
| Replied | Lead replied |
| Qualified | Real fit or useful problem confirmed |
| Demo / Call | Meeting, demo, or async walkthrough scheduled |
| Nurture | Good contact but no immediate need |
| Closed | Not relevant, no response after follow-up, or not a fit |
