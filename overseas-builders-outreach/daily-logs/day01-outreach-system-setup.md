# Day 1 — Outreach System Setup

Date:

## Goal

Set up the overseas builders outreach system before sending messages.

The goal is to avoid random cold DMs. We need a clear target profile, lead scoring model, outreach angles, and follow-up workflow.

## Ideal Lead Profile

Best fit:

- building AI agents that use tools
- running browser, shell, file, code, or API tools
- discussing production reliability, sandboxing, runtime, observability, or model routing
- reachable on X, LinkedIn, GitHub, Peerlist, Discord, or email
- likely to benefit from sandboxed runtime or infrastructure support

## Lead Sources

- X search
- LinkedIn search
- GitHub repositories
- Peerlist posts
- Hacker News launches
- Reddit technical threads
- Dev.to posts
- hackathon project galleries
- AI agent newsletters

## First Outreach Angles To Test

1. Runtime reliability
2. Sandboxed tool execution
3. Production agent checklist
4. Builder support / architecture review

## Today’s Setup Checklist

- [ ] Create first 20-lead list.
- [ ] Score each lead.
- [ ] Pick top 5.
- [ ] Prepare personalized first messages.
- [ ] Send only after manual review.
- [ ] Record replies and next actions.

## Notes

Do not optimize for message volume on Day 1.

Optimize for relevance.
