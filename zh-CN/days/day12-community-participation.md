# Day 12 — 不做链接 spam 的社区参与

日期: TBD

阶段: 第 2 周 — 分发和日常运营

状态: 计划中

## 目标

找到相关开发者讨论，先贡献有用观点，再考虑分享链接。

## 计划工作

- 找到 3-5 个相关讨论。
- 只在 SandBase 的视角真正有帮助时回复。
- 不把链接作为第一动作。
- 记录重复问题、异议和用户用词。
- 把好问题转成第 3 周内容输入。

## 候选社区

- Hacker News
- AI agents、本地模型、开发者工具相关 Reddit 社区
- Indie Hackers
- 相关 GitHub discussions 或 issues
- AI infra builder 的 X 和 LinkedIn threads

## 使用工具

| 工具 | 作用 |
|------|------|
| Codex | 研究、起草回复、总结问题 |
| Browser | 阅读和参与，公开操作前确认 |
| Discord / X / LinkedIn | 日常社区触点 |
