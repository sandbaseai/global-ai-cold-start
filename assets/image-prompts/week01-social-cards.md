# Week 1 Social Image Prompts

These prompts are designed for image generation tools such as image2 / imagegen.

Use them to create simple, readable visual cards for X, LinkedIn, and blog embeds.

Recommended export:

- 1600x900 for LinkedIn and blog embeds
- 1200x675 for X
- clean dark or light technical editorial style
- avoid tiny text
- keep copy short and readable

## Style System

Use this shared style for all cards:

```text
Use case: productivity-visual
Asset type: social media image for X and LinkedIn
Style/medium: polished editorial tech infographic, clean UI-like layout, high readability
Scene/backdrop: abstract infrastructure workspace, subtle grid, browser/code/community panels
Composition/framing: centered title, 3-5 visual blocks, generous spacing, no clutter
Lighting/mood: calm, credible, technical, founder build-in-public energy
Color palette: off-white background with black text, small teal accents, muted gray panels
Text constraints: large readable typography, no tiny paragraphs, no misspellings
Avoid: stock-photo people, cartoon mascots, hype style, neon cyberpunk, excessive gradients, fake logos, unreadable UI text
```
