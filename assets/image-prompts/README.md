# Image Prompts

This folder stores reusable image prompts for the public growth journal.

The goal is to make the project easier to share on:

- X
- LinkedIn
- Blog posts
- README embeds
- weekly recap posts

## Current Prompt Sets

- [Week 1 social cards](week01-social-cards.md)

## Production Rule

Every generated image should be:

- readable without zooming
- understandable by a beginner
- visually consistent with SandBase's infrastructure positioning
- useful as a standalone social post image
- free of private account details, emails, tokens, or internal UI screenshots

Avoid:

- tiny text
- fake product claims
- over-designed AI art
- stock-photo style
- real third-party platform logos unless explicitly allowed
