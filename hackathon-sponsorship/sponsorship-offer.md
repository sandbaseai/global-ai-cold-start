# SandBase Hackathon Support Offer

SandBase helps teams build more realistic AI agents during hackathons.

## Who This Is For

- hackathon organizers
- AI builder communities
- university AI clubs
- developer communities
- online AI agent challenges

## What SandBase Can Provide

- credits for selected participants
- sandboxed runtime access
- model/API support
- starter templates for tool-using agents
- technical office hours
- review sessions for production-agent architecture
- prize support for agent infrastructure tracks

## Suggested Prize Track

```text
Best Production-Ready Agent
```

Judging criteria:

- uses tools safely
- has clear runtime boundaries
- logs important agent actions
- handles failure or retries
- avoids hardcoded secrets
- can explain what happens when the agent runs

## Why It Helps Builders

Most hackathon agent demos focus on the model and UI.

SandBase helps teams think about the production layer:

- where tools run
- how file/browser/shell access is isolated
- how model calls and tool calls are tracked
- how agent workloads can be bounded by time, cost, and permissions

## Organizer Message

```text
SandBase can support AI agent teams with runtime credits, starter templates, and office hours focused on production-ready agents.

The goal is to help builders move beyond prompt demos and ship agents that use tools with clearer boundaries.
```
