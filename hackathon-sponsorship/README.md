# Hackathon Sponsorship

Part 3 of Global AI Cold Start.

This section records how SandBase approaches AI hackathons and builder events as an infrastructure sponsor.

Why hackathons:

- they gather active AI builders
- they create real agent use cases quickly
- they reveal tooling and runtime pain points
- they create natural developer relationships without cold selling

Potential offers:

- sandboxed runtime credits
- model/API credits
- starter templates
- technical office hours
- prize tracks for production-ready agents
- support for teams building tool-using agents

The goal is to help AI builders ship more realistic agent projects while creating a natural path from hackathon usage to long-term developer adoption.

What will be included:

- sponsorship offer page
- target event list
- organizer outreach templates
- prize track ideas
- technical support package
- post-event follow-up plan
