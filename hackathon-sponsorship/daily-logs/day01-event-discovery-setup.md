# Day 1 — Event Discovery Setup

Date:

## Goal

Set up the hackathon sponsorship discovery workflow.

The goal is to find events where SandBase can help builders ship better AI agent projects, not just buy logo placement.

## Ideal Event Profile

Best fit:

- AI agents, AI infra, devtools, automation, or open-source theme
- global or remote participation
- teams are building real technical projects
- organizers accept technical sponsors or community partners
- event happens within the next 1-4 months
- prize tracks or workshops are possible

## Event Sources

- Devpost
- Lablab.ai
- DoraHacks
- HackQuest
- Eventbrite
- Meetup
- university AI clubs
- developer communities
- AI newsletters
- X / LinkedIn event posts

## Today’s Setup Checklist

- [ ] Find 10 candidate events.
- [ ] Score each event.
- [ ] Pick top 3.
- [ ] Prepare organizer outreach.
- [ ] Define SandBase support package.
- [ ] Record follow-up dates.

## Notes

The best sponsorship angle is technical support for tool-using agents.

Avoid events where the only offer is a paid logo placement.
